Project 1: Home Network & Connectivity Analysis

Objective

To test and analyze my home network connectivity using Windows command-line tools, and to identify how devices communicate on a local network.

Tools Used

  • Command Prompt (ipconfig, ping)
  • Windows Network Settings

Tasks Performed

  • Identified IP address, Default Gateway, and DNS Server using ipconfig
  • Tested internet connectivity using ping
  • Verified packet response and network stability

Key Findings

The device successfully communicated with the network and external servers, with no packet loss and stable response times.

Conclusion

This project demonstrates core networking knowledge and basic troubleshooting skills.
