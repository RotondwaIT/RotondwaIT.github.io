Project 3: PC Hardware Components & Upgrade Plan

Objective

To identify key hardware components of a computer and create an upgrade and maintenance plan for improved system performance.

Tools Used

  • System Information (msinfo32)
  • Task Manager (Performance tab)

Tasks Performed

  • Identified CPU, RAM, and storage type
  • Checked performance using Task Manager
  • Documented current specifications

Upgrade Plan

Recommendations include adding more RAM for multitasking and replacing the HDD with a larger SSD for faster performance and reliability.

Conclusion

This project demonstrates hardware understanding and the ability to assess and plan computer system improvements, which are key skills in IT Support.
