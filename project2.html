Project 2: Windows System Diagnostics & IT Support Tools

Objective

To analyze a Windows computer using built-in system diagnostic tools and demonstrate IT support skills.

Tools Used

  • Task Manager
  • Device Manager
  • System Information (msinfo32)
  • Command Prompt (systeminfo)

Tasks Performed

  • Monitored performance and processes using Task Manager
  • Checked hardware and drivers using Device Manager
  • Collected system details with msinfo32 and systeminfo

Key Findings

The analysis showed stable performance, active hardware components, and a correctly configured Windows system.

Conclusion

This project demonstrates IT support capability in troubleshooting, monitoring, and system diagnostics.
